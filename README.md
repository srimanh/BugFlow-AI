# 🐞 BugFlow – AI-Assisted Bug Tracker

> 🔐 Role-Based Bug Management + 🤖 AI Root Cause Suggestions  
> Built with **Spring Boot**, **PostgreSQL**, and **OpenAI**.

---

## 🚀 Overview

**BugFlow** is an intelligent bug tracking system that helps QA testers and developers manage, track, and resolve bugs in a structured, scalable, and AI-assisted environment.

### ✨ Core Features
- 🐞 **Bug Reporting and Assignment**
- 🤖 **AI Root Cause Suggestion** (via OpenAI)
- 👥 **Role-based Access Control** (QA, Developer, Manager)
- 🧠 **Similar Bug Finder** by keyword
- 📊 **Module-wise Tracking**
- 🔒 **JWT Authentication + Secure APIs**

---

## 🛠️ Tech Stack

| Layer         | Tech                                  |
|---------------|----------------------------------------|
| 🧠 Backend     | Spring Boot, Java 17                  |
| 🗃️ Database    | PostgreSQL                            |
| 🔐 Security    | Spring Security + BCrypt + JWT        |
| 🤖 AI Service  | OpenAI API (for suggestions)          |
| 📄 Docs        | Swagger (API documentation)           |
